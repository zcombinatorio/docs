title: "Get All Proposals"
description: "GET /dao/proposals/all - Get all proposals from all DAOs"
---

## Overview

Returns all proposals from all DAOs in the system, sorted by creation date (newest first). Useful for building a global markets view.

<CodeGroup>

```bash curl
curl https://api.zcombinator.io/dao/proposals/all
```

```javascript fetch
const response = await fetch('https://api.zcombinator.io/dao/proposals/all');
const { proposals } = await response.json();
```

```python requests
import requests

response = requests.get('https://api.zcombinator.io/dao/proposals/all')
proposals = response.json()['proposals']
```

</CodeGroup>

## Response

<ResponseField name="proposals" type="array">
  Array of proposal objects with DAO metadata
</ResponseField>

### Proposal Object

All fields from [List Proposals](/api-reference/analytics/list-proposals) plus:

<ResponseField name="daoPda" type="string">
  The DAO's Program Derived Address
</ResponseField>

<ResponseField name="daoName" type="string">
  The DAO's name
</ResponseField>

<ResponseField name="tokenMint" type="string">
  The DAO's base token mint
</ResponseField>

<ResponseField name="tokenIcon" type="string | null">
  Token icon URL from metadata
</ResponseField>

## Notes

- Proposals in `Setup` state are filtered out
- Results include DAO metadata for each proposal
- Sorted by creation date (newest first)
- Token icons and decimals are fetched from on-chain metadata
- This endpoint may take longer to respond as it queries all DAOs
