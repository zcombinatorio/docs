title: "Get DAO"
description: "GET /dao/:daoPda - Get detailed information about a specific DAO"
---

## Overview

Returns detailed information about a specific DAO, including its stats, proposers, and child DAOs (if applicable).

<CodeGroup>

```bash curl
curl https://api.zcombinator.io/dao/Dao1xgp...ABC
```

```javascript fetch
const daoPda = 'Dao1xgp...ABC';
const response = await fetch(`https://api.zcombinator.io/dao/${daoPda}`);
const dao = await response.json();
```

```python requests
import requests

dao_pda = "Dao1xgp...ABC"
response = requests.get(f'https://api.zcombinator.io/dao/{dao_pda}')
dao = response.json()
```

</CodeGroup>

## Path Parameters

<ParamField path="daoPda" type="string" required>
  The DAO's Program Derived Address
</ParamField>

## Response

All fields from [List DAOs](/api-reference/analytics/list-daos) plus:

<ResponseField name="proposers" type="array">
  Array of whitelisted proposer objects
</ResponseField>

<ResponseField name="children" type="array">
  Array of child DAO objects (only for parent DAOs)
</ResponseField>

### Proposer Object

<ResponseField name="id" type="number">
  Internal proposer ID
</ResponseField>

<ResponseField name="dao_id" type="number">
  Parent DAO's internal ID
</ResponseField>

<ResponseField name="proposer_wallet" type="string">
  Whitelisted wallet address
</ResponseField>

<ResponseField name="added_by" type="string">
  Wallet that added this proposer
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO timestamp
</ResponseField>
