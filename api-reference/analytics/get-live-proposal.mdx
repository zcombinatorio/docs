---

## Overview

Returns the currently active (Pending status) proposal for a DAO. Returns 404 if no live proposal exists.

<CodeGroup>

```bash curl
curl https://api.zcombinator.io/dao/Dao1xgp...ABC/proposal/live
```

```javascript fetch
const daoPda = 'Dao1xgp...ABC';
const response = await fetch(`https://api.zcombinator.io/dao/${daoPda}/proposal/live`);
const proposal = await response.json();
```

```python requests
import requests

dao_pda = "Dao1xgp...ABC"
response = requests.get(f'https://api.zcombinator.io/dao/{dao_pda}/proposal/live')
proposal = response.json()
```

</CodeGroup>

## Path Parameters

<ParamField path="daoPda" type="string" required>
  The DAO's Program Derived Address
</ParamField>

## Response

<ResponseField name="id" type="number">
  Proposal ID within the moderator
</ResponseField>

<ResponseField name="proposalPda" type="string">
  The Proposal's Program Derived Address
</ResponseField>

<ResponseField name="title" type="string">
  Proposal title from IPFS metadata
</ResponseField>

<ResponseField name="description" type="string">
  Proposal description
</ResponseField>

<ResponseField name="options" type="string[]">
  Array of option labels
</ResponseField>

<ResponseField name="status" type="string">
  Always `Pending` for live proposals
</ResponseField>

<ResponseField name="winningIndex" type="null">
  Always null for live proposals
</ResponseField>

<ResponseField name="numOptions" type="number">
  Number of options in the proposal
</ResponseField>

<ResponseField name="createdAt" type="number">
  Unix timestamp in milliseconds
</ResponseField>

<ResponseField name="endsAt" type="number">
  End time in milliseconds
</ResponseField>

<ResponseField name="warmupEndsAt" type="number">
  Warmup end time in milliseconds
</ResponseField>

<ResponseField name="moderator" type="string">
  Moderator PDA
</ResponseField>

<ResponseField name="creator" type="string">
  Proposal creator wallet
</ResponseField>

<ResponseField name="vault" type="string">
  Vault address
</ResponseField>

<ResponseField name="baseMint" type="string">
  Base token mint address
</ResponseField>

<ResponseField name="quoteMint" type="string">
  Quote token mint address
</ResponseField>

<ResponseField name="baseDecimals" type="number">
  Base token decimals
</ResponseField>

<ResponseField name="quoteDecimals" type="number">
  Quote token decimals
</ResponseField>

<ResponseField name="pools" type="string[]">
  Array of pool addresses for each option
</ResponseField>

<ResponseField name="metadataCid" type="string | null">
  IPFS CID for metadata
</ResponseField>

<ResponseField name="daoPda" type="string">
  The DAO's PDA
</ResponseField>

<ResponseField name="config" type="object">
  Proposal configuration
</ResponseField>

## Notes

- Only returns proposals in `Pending` state
- Searches from newest to oldest for efficiency
- Child DAOs inherit the parent's moderator for proposal lookups
