title: "Get Proposal"
description: "GET /dao/proposal/:proposalPda - Get detailed information about a specific proposal"

## Overview

Returns detailed information about a specific proposal by its PDA. Fetches data directly from on-chain and IPFS.

<CodeGroup>

```bash curl
curl https://api.zcombinator.io/dao/proposal/Prop1xgp...ABC
```

```javascript fetch
const proposalPda = 'Prop1xgp...ABC';
const response = await fetch(`https://api.zcombinator.io/dao/proposal/${proposalPda}`);
const proposal = await response.json();
```

```python requests
import requests

proposal_pda = "Prop1xgp...ABC"
response = requests.get(f'https://api.zcombinator.io/dao/proposal/{proposal_pda}')
proposal = response.json()
```

</CodeGroup>

## Path Parameters

<ParamField path="proposalPda" type="string" required>
  The Proposal's Program Derived Address
</ParamField>

## Response

<ResponseField name="id" type="number">
  Proposal ID within the moderator
</ResponseField>

<ResponseField name="proposalPda" type="string">
  The Proposal's Program Derived Address
</ResponseField>

<ResponseField name="title" type="string">
  Proposal title from IPFS metadata
</ResponseField>

<ResponseField name="description" type="string">
  Proposal description
</ResponseField>

<ResponseField name="options" type="string[]">
  Array of option labels
</ResponseField>

<ResponseField name="status" type="string">
  Proposal status: `Setup`, `Pending`, or `Resolved`
</ResponseField>

<ResponseField name="winningIndex" type="number | null">
  Index of winning option (only for resolved proposals)
</ResponseField>

<ResponseField name="numOptions" type="number">
  Number of options in the proposal
</ResponseField>

<ResponseField name="createdAt" type="number">
  Unix timestamp in milliseconds
</ResponseField>

<ResponseField name="endsAt" type="number">
  End time in milliseconds
</ResponseField>

<ResponseField name="warmupEndsAt" type="number">
  Warmup end time in milliseconds
</ResponseField>

<ResponseField name="moderator" type="string">
  Moderator PDA
</ResponseField>

<ResponseField name="creator" type="string">
  Proposal creator wallet
</ResponseField>

<ResponseField name="vault" type="string">
  Vault address
</ResponseField>

<ResponseField name="baseMint" type="string">
  Base token mint address
</ResponseField>

<ResponseField name="quoteMint" type="string">
  Quote token mint address
</ResponseField>

<ResponseField name="pools" type="string[]">
  Array of pool addresses for each option
</ResponseField>

<ResponseField name="metadataCid" type="string | null">
  IPFS CID for metadata
</ResponseField>

<ResponseField name="daoPda" type="string | null">
  The DAO's PDA (from metadata)
</ResponseField>

<ResponseField name="config" type="object">
  Proposal configuration object
</ResponseField>

### Config Object

<ResponseField name="config.length" type="number">
  Proposal duration in seconds
</ResponseField>

<ResponseField name="config.warmupDuration" type="number">
  Warmup period in seconds
</ResponseField>

<ResponseField name="config.marketBias" type="number">
  Market bias setting
</ResponseField>

<ResponseField name="config.fee" type="number">
  Fee in basis points
</ResponseField>

```json
    }
    ```
  </Accordion>

## Notes

- This endpoint fetches directly from the Solana blockchain
- IPFS metadata is fetched asynchronously; if unavailable, default values are used
- The `pools` array contains addresses for each option's conditional token AMM
