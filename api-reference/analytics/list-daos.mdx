---
title: "List DAOs"
description: "GET /dao - List all DAOs with optional filtering"
---

## Overview

Returns a list of all DAOs registered in the system. Supports filtering by type and owner.

<CodeGroup>

```bash curl
curl https://api.zcombinator.io/dao
```

```javascript fetch
const response = await fetch('https://api.zcombinator.io/dao');
const { daos } = await response.json();
```

```python requests
import requests

response = requests.get('https://api.zcombinator.io/dao')
daos = response.json()['daos']
```

</CodeGroup>

## Query Parameters

<ParamField query="type" type="string">
  Filter by DAO type: `parent` or `child`
</ParamField>

<ParamField query="owner" type="string">
  Filter by owner wallet address
</ParamField>

<ParamField query="limit" type="number">
  Maximum number of results to return
</ParamField>

<ParamField query="offset" type="number">
  Number of results to skip (for pagination)
</ParamField>

## Response

<ResponseField name="daos" type="array">
  Array of DAO objects
</ResponseField>

### DAO Object

<ResponseField name="id" type="number">
  Internal DAO ID
</ResponseField>

<ResponseField name="dao_pda" type="string">
  The DAO's Program Derived Address
</ResponseField>

<ResponseField name="dao_name" type="string">
  DAO name
</ResponseField>

<ResponseField name="moderator_pda" type="string">
  Moderator PDA for proposals
</ResponseField>

<ResponseField name="owner_wallet" type="string">
  DAO owner's wallet address
</ResponseField>

<ResponseField name="admin_wallet" type="string">
  Managed admin wallet address
</ResponseField>

<ResponseField name="token_mint" type="string">
  Base token mint address
</ResponseField>

<ResponseField name="quote_mint" type="string">
  Quote token mint address
</ResponseField>

<ResponseField name="pool_address" type="string">
  Meteora pool address
</ResponseField>

<ResponseField name="pool_type" type="string">
  Pool type (`dlmm` or `cpAmm`)
</ResponseField>

<ResponseField name="treasury_vault" type="string">
  Squads treasury vault address
</ResponseField>

<ResponseField name="mint_vault" type="string">
  Squads mint authority vault address
</ResponseField>

<ResponseField name="dao_type" type="string">
  `parent` or `child`
</ResponseField>

<ResponseField name="withdrawal_percentage" type="number">
  Liquidity withdrawal percentage for proposals
</ResponseField>

<ResponseField name="icon" type="string | null">
  Token icon URL from metadata
</ResponseField>

<ResponseField name="token_decimals" type="number">
  Base token decimals
</ResponseField>

<ResponseField name="quote_decimals" type="number">
  Quote token decimals
</ResponseField>

<ResponseField name="stats" type="object">
  DAO statistics including proposalCount
</ResponseField>

## Filtering Examples

### Get Parent DAOs Only

```bash
curl "https://api.zcombinator.io/dao?type=parent"
```

### Get DAOs by Owner

```bash
curl "https://api.zcombinator.io/dao?owner=9WzDXwBbmkg8ZTbNMqUxvQRAyrZzDsGYdLVL9zYtAWWM"
```

### Pagination

```bash
curl "https://api.zcombinator.io/dao?limit=10&offset=0"
```
