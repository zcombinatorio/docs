---

## Overview

Returns all proposals for a specific DAO, sorted by creation date (newest first). Fetches data from on-chain and IPFS.

<CodeGroup>

```bash curl
curl https://api.zcombinator.io/dao/Dao1xgp...ABC/proposals
```

```javascript fetch
const daoPda = 'Dao1xgp...ABC';
const response = await fetch(`https://api.zcombinator.io/dao/${daoPda}/proposals`);
const { proposals } = await response.json();
```

```python requests
import requests

dao_pda = "Dao1xgp...ABC"
response = requests.get(f'https://api.zcombinator.io/dao/{dao_pda}/proposals')
proposals = response.json()['proposals']
```

</CodeGroup>

## Path Parameters

<ParamField path="daoPda" type="string" required>
  The DAO's Program Derived Address
</ParamField>

## Response

<ResponseField name="proposals" type="array">
  Array of proposal objects
</ResponseField>

### Proposal Object

<ResponseField name="id" type="number">
  Proposal ID within the moderator
</ResponseField>

<ResponseField name="proposalPda" type="string">
  The Proposal's Program Derived Address
</ResponseField>

<ResponseField name="title" type="string">
  Proposal title from IPFS metadata
</ResponseField>

<ResponseField name="description" type="string">
  Proposal description from IPFS metadata
</ResponseField>

<ResponseField name="options" type="string[]">
  Array of option labels
</ResponseField>

<ResponseField name="status" type="string">
  Proposal status: `Setup`, `Pending`, or `Resolved`
</ResponseField>

<ResponseField name="winningIndex" type="number | null">
  Index of winning option (only for resolved proposals)
</ResponseField>

<ResponseField name="vault" type="string">
  Vault address for the proposal
</ResponseField>

<ResponseField name="createdAt" type="number">
  Unix timestamp in milliseconds
</ResponseField>

<ResponseField name="endsAt" type="number | null">
  End time in milliseconds
</ResponseField>

<ResponseField name="finalizedAt" type="number | null">
  Finalization time in milliseconds
</ResponseField>

<ResponseField name="metadataCid" type="string | null">
  IPFS CID for proposal metadata
</ResponseField>

<ResponseField name="marketBias" type="number">
  Market bias configuration
</ResponseField>

<ResponseField name="baseDecimals" type="number">
  Base token decimals
</ResponseField>

<ResponseField name="quoteDecimals" type="number">
  Quote token decimals
</ResponseField>

## Notes

- Proposals in `Setup` state are filtered out
- Results are sorted by proposal ID (newest first)
- IPFS metadata is fetched asynchronously; if unavailable, default values are used
