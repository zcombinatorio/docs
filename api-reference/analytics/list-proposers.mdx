---

## Overview

Returns the list of whitelisted proposers for a specific DAO.

<CodeGroup>

```bash curl
curl https://api.zcombinator.io/dao/Dao1xgp...ABC/proposers
```

```javascript fetch
const daoPda = 'Dao1xgp...ABC';
const response = await fetch(`https://api.zcombinator.io/dao/${daoPda}/proposers`);
const { owner, proposers } = await response.json();
```

```python requests
import requests

dao_pda = "Dao1xgp...ABC"
response = requests.get(f'https://api.zcombinator.io/dao/{dao_pda}/proposers')
data = response.json()
```

</CodeGroup>

## Path Parameters

<ParamField path="daoPda" type="string" required>
  The DAO's Program Derived Address
</ParamField>

## Response

<ResponseField name="owner" type="string">
  The DAO owner's wallet address
</ResponseField>

<ResponseField name="proposers" type="array">
  Array of proposer objects
</ResponseField>

### Proposer Object

<ResponseField name="id" type="number">
  Internal proposer ID
</ResponseField>

<ResponseField name="dao_id" type="number">
  Parent DAO's internal ID
</ResponseField>

<ResponseField name="proposer_wallet" type="string">
  Whitelisted wallet address
</ResponseField>

<ResponseField name="added_by" type="string">
  Wallet that added this proposer
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO timestamp when the proposer was added
</ResponseField>

## Notes

- The DAO owner is always authorized to create proposals, regardless of whitelist status
- Proposers can also qualify via token threshold (see [Set Proposer Token Threshold](/api-reference/dao/set-threshold))
