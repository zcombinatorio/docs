## Overview

Reserves a managed admin wallet that a client can use when creating a DAO on-chain in their own transaction. This enables clients to bundle DAO creation with other instructions (token launch, pool creation, LP transfer, etc.) in a single transaction.

The DAO is not active until you call [`POST /dao/finalize-reserved`](/api-reference/dao/finalize-reserved) after your on-chain transaction confirms.

<Info>
This endpoint uses API key authentication, not wallet signature authentication. Contact the team to obtain an API key.
</Info>

## Flow

```
1. POST /dao/reserve-admin     → get admin_wallet + dao_id
2. Build on-chain transaction:
   - Create token + pool
   - Create DAO (with yourself as temp admin)
   - transferAdmin to the reserved admin_wallet
   - Transfer LP to admin_wallet
   - Transfer mint authority
3. Submit transaction
4. POST /dao/finalize-reserved  → activate the DAO
```

## Request Parameters

<ParamField body="api_key" type="string" required>
  API key for authentication
</ParamField>

<ParamField body="name" type="string" required>
  DAO name (max 32 characters). Must be unique.
</ParamField>

<ParamField body="token_mint" type="string" required>
  Governance token mint address
</ParamField>

<ParamField body="pool_address" type="string" required>
  Meteora pool address
</ParamField>

<ParamField body="pool_type" type="string" required>
  Pool type: `"damm"` or `"dlmm"`
</ParamField>

<ParamField body="quote_mint" type="string" required>
  Quote token mint address (e.g., USDC or SOL mint)
</ParamField>

<ParamField body="treasury_cosigner" type="string" required>
  Treasury co-signer wallet address
</ParamField>

<ParamField body="owner_wallet" type="string" required>
  DAO owner wallet address. Automatically added to the proposer whitelist.
</ParamField>

## Response

<ResponseField name="admin_wallet" type="string">
  The managed admin wallet address. Use this as the target for `transferAdmin` in your on-chain transaction.
</ResponseField>

<ResponseField name="admin_key_idx" type="number">
  Internal key index for the admin wallet
</ResponseField>

<ResponseField name="dao_id" type="number">
  The DAO's ID. Pass this to `POST /dao/finalize-reserved` after on-chain confirmation.
</ResponseField>

### Success Response

```json
{
  "admin_wallet": "AdminWallet1xgp...ABC",
  "admin_key_idx": 5,
  "dao_id": 42
}
```
