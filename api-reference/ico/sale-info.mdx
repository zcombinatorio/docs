## Overview

Retrieves detailed information about an ICO sale, including total tokens for sale, tokens sold, SOL raised, and sale status.

<CodeGroup>

```bash curl
curl -X GET "https://api.zcombinator.io/ico/EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v"
```

```javascript fetch
const tokenAddress = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v";

const response = await fetch(
  `https://api.zcombinator.io/ico/${tokenAddress}`
);

const saleInfo = await response.json();
```

```python requests
import requests

token_address = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v"

response = requests.get(
    f'https://api.zcombinator.io/ico/{token_address}'
)

sale_info = response.json()
```

</CodeGroup>

## URL Parameters

<ParamField path="tokenAddress" type="string" required>
  The Solana token mint address of the ICO
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the ICO sale
</ResponseField>

<ResponseField name="token_address" type="string">
  The token mint address
</ResponseField>

<ResponseField name="creator_wallet" type="string">
  The wallet address of the ICO creator
</ResponseField>

<ResponseField name="escrow_pub_key" type="string">
  Public key of the escrow account holding tokens
</ResponseField>

<ResponseField name="token_metadata_url" type="string">
  URL to the token's metadata JSON
</ResponseField>

<ResponseField name="total_tokens_for_sale" type="string">
  Total number of tokens available in the ICO (as string for large numbers)
</ResponseField>

<ResponseField name="tokens_sold" type="string">
  Number of tokens already sold
</ResponseField>

<ResponseField name="total_sol_raised" type="string">
  Total SOL raised in lamports
</ResponseField>

<ResponseField name="token_price_sol" type="string">
  Price per token in SOL
</ResponseField>

<ResponseField name="vault_token_account" type="string">
  Token account where 50% of purchased tokens are sent
</ResponseField>

<ResponseField name="treasury_wallet" type="string">
  Wallet receiving SOL from purchases
</ResponseField>

<ResponseField name="treasury_sol_amount" type="string">
  Amount of SOL allocated for treasury
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the ICO: `active` or `finalized`
</ResponseField>

<ResponseField name="token_decimals" type="number">
  Number of decimals for the token (default: 9)
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the ICO was created
</ResponseField>

## ICO Sale Modes

The ICO operates in two distinct modes that never overlap:

<CardGroup cols={2}>
  <Card title="Active Mode" icon="shopping-cart">
    - Users can purchase tokens
    - Claiming is blocked
    - Overselling protection is enforced
  </Card>
  <Card title="Finalized Mode" icon="gift">
    - Purchasing is blocked
    - Users can claim their 50% escrow tokens
    - No overselling checks needed
  </Card>
</CardGroup>

## Rate Limiting

This endpoint is subject to rate limiting:
- **30 requests per IP** per 1-minute window
- Returns HTTP 429 when limit exceeded
